# 🏎️ Python Car Simulator

A simple terminal-based car game where you race, earn credits, upgrade your car, and try not to go broke doing it.

## 🚀 Features
	•	Race system (win or lose 💀)
	•	Fuel system (no fuel = no racing)
	•	Credit system 💰
	•	Upgrade system (performance actually matters)
	•	Car stats display

## 🎮 How to play

## Run the script and:
	1.	Build your car
	2.	Race to earn credits
	3.	Refuel when needed
	4.	Upgrade your car
	5.	Repeat until you’re rich or broke
